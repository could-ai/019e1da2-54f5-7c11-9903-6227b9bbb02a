# مصطفى همو - السيرة الذاتية

تطبيق سيرة ذاتية تفاعلي للأستاذ مصطفى همو، أستاذ مادة اللغة العربية والتربية الإسلامية.

## المميزات
- تصميم متجاوب يعمل على جميع الأجهزة (الموبايل، التابلت، سطح المكتب، الويب).
- واجهة مستخدم باللغة العربية (RTL).
- عرض المعلومات الشخصية وطرق التواصل.
- أقسام مخصصة للنبذة الشخصية، الخبرات المهنية، المؤهلات العلمية، والمهارات.

## التقنيات المستخدمة
- Flutter
- Dart

## كيفية التشغيل
1. تأكد من تثبيت Flutter على جهازك.
2. قم بتنفيذ الأمر التالي لتحميل الاعتماديات:
   ```bash
   flutter pub get
   ```
3. قم بتشغيل التطبيق:
   ```bash
   flutter run
   ```
